## Introduction
A computer only understands binary(0s and 1s). Programming languages enable humans to write logical, and readable instructions that are translated to bonary for the computer to execute.
They are used for web development, mobile applications for phones and tablets,Cyber security to find vulnerabilities and build security tools, Artificial Intelligence and machine learning to teach computer to learn and make decisions, database management for storing queries and managing data among others.
